[Link to site](https://admin-bit-sigma.vercel.app/)


Реализовать первый экран пользовательского интерфейса панели администратора, который будет отображать список пользователей с функциями поиска в таблице и сортировки по количеству токенов.
По нажатию на пользователя должен отображаться всплывающий боковой элемент (drawer) с графиком расходов за последние 24 часа и перечнем транзакций данного пользователя.
